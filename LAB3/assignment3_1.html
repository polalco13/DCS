Explanation of Cubic Lagrange Interpolation

Mathematical Foundation

Lagrange interpolation is a method for finding a polynomial that passes through a given set of points. For n points, we can construct a polynomial of degree n-1 that passes through all these points.

For our case with 4 points (P₀, P₁, P₂, P₃), we construct a cubic (degree 3) polynomial.

Lagrange Basis Polynomials

For 4 points with parameter values t₀, t₁, t₂, t₃, the Lagrange basis polynomials are: 

L₀(t) = ((t-t₁)(t-t₂)(t-t₃))/((t₀-t₁)(t₀-t₂)(t₀-t₃))
L₁(t) = ((t-t₀)(t-t₂)(t-t₃))/((t₁-t₀)(t₁-t₂)(t₁-t₃))
L₂(t) = ((t-t₀)(t-t₁)(t-t₃))/((t₂-t₀)(t₂-t₁)(t₂-t₃))
L₃(t) = ((t-t₀)(t-t₁)(t-t₂))/((t₃-t₀)(t₃-t₁)(t₃-t₂))

The Lagrange polynomial is then: P(t) = L₀(t)P₀ + L₁(t)P₁ + L₂(t)P₂ + L₃(t)P₃

Parameterization Methods

1. Uniform parameter values: The parameter values are evenly spaced between 0 and 1: 

t₀ = 0, t₁ = 1/3, t₂ = 2/3, t₃ = 1

2. Non-uniform parameter values (distance-based): The parameter values are based on the distances between consecutive points: 

t₀ = 0
t₁ = d₀₁/(d₀₁+d₁₂+d₂₃)
t₂ = (d₀₁+d₁₂)/(d₀₁+d₁₂+d₂₃)
t₃ = 1

Where dij is the Euclidean distance between points Pi and Pj.

Visual Comparison

Observe the difference: With uniform parameterization, the curve may bunch up or stretch out unevenly. With non-uniform parameterization, the curve distribution often appears more balanced and natural, especially when points are unevenly spaced.
